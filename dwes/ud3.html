Criterios de Evaluación Descripción Ejemplo en JSTL
a) Mecanismos de decisión El alumno debe ser capaz de implementar estructuras de control de decisión. <c:choose>
<c:when test="${condicion}">...
<c:otherwise>...
b) Bucles El alumno debe implementar bucles y verificar su correcto funcionamiento. <c:forEach var="item" items="${lista}">...
c) Matrices (arrays) El alumno debe implementar y manipular matrices o listas para almacenar y recuperar conjuntos de datos. <c:forEach var="item" items="${arrayItems}"> ${item} </c:forEach>
d) Creación y uso de funciones El alumno debe ser capaz de crear y usar funciones dentro de JSTL. ${fn:toUpperCase('ejemplo')} (Convierte el texto 'ejemplo' a mayúsculas)
e) Uso de formularios web El alumno debe implementar formularios web para recopilar datos del usuario. <form action="procesar.jsp">
<input type="text" name="nombre" />
</form>
f) Recuperación de información del formulario El alumno debe recuperar y procesar la información proporcionada en el formulario. ${param.nombre} (Recupera el valor del campo 'nombre' del formulario)
g) Comentarios en el código El alumno debe añadir comentarios relevantes en el código para mejorar su comprensión. <!-- Este es un comentario en JSP -->
