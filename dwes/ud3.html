Sistema de Registro de Estudiantes

Desarrolla un sistema web simple utilizando Spring Boot que permita:

  • Registrar estudiantes con los siguientes datos: nombre, edad, grado y lista de materias inscritas (usando un array).
  • Listar todos los estudiantes registrados.
  • Buscar estudiantes por nombre.
  • Mostrar el promedio de edad de los estudiantes registrados.
  • Añadir comentarios explicativos en el código.

Criterio de Evaluación Insatisfactorio Básico Competente Sobresaliente
a) Mecanismos de decisión No utiliza mecanismos de decisión Utiliza mecanismos de decisión de forma básica y en pocas ocasiones Utiliza mecanismos de decisión de forma adecuada Utiliza mecanismos de decisión de forma experta y eficiente
b) Uso de bucles No utiliza bucles Utiliza bucles, pero no verifica su funcionamiento Utiliza y verifica bucles adecuadamente Utiliza bucles de manera óptima y verifica su funcionamiento exhaustivamente
c) Uso de arrays No utiliza arrays Utiliza arrays de manera limitada Utiliza arrays de manera correcta para almacenar y recuperar datos Utiliza arrays de manera óptima, explorando diversas funciones y métodos sobre los mismos
d) Creación y uso de funciones No define ni utiliza funciones Define y utiliza funciones básicas sin parámetros Define y utiliza funciones, incluyendo el uso de parámetros Define y utiliza funciones avanzadas, con parámetros y retornos complejos
e) Interacción mediante formularios web No utiliza formularios web Utiliza formularios web sin validación de datos Utiliza formularios web con validación básica de datos Utiliza formularios web con validación avanzada de datos y UX amigable
f) Métodos para recuperar información del formulario No emplea métodos para recuperar información Emplea métodos básicos sin validación para recuperar información Emplea métodos con validaciones básicas para recuperar información Emplea métodos de forma óptima, incluyendo validaciones y gestionando posibles errores
g) Añadir comentarios al código No añade comentarios al código Añade algunos comentarios, pero no son claros ni descriptivos Añade comentarios claros y descriptivos en las partes clave del código Añade comentarios claros, descriptivos y concisos en todas las partes necesarias del código
