Posibles preguntas de Evaluación

  1. [Criterio a] ¿Cuál es la diferencia fundamental entre ejecutar un script JavaScript en el lado del cliente y ejecutar un código Java en el servidor utilizando Spring Boot? Proporciona un ejemplo para cada uno.
  2. [Criterio b] ¿Cómo beneficia la integración de tecnologías cliente-servidor al desarrollo de aplicaciones web en términos de rendimiento y experiencia de usuario? Proporcione ejemplos específicos utilizando Spring Boot y JavaScript.
  3. [Criterio c] Identifique y describa brevemente dos dependencias en Spring Boot que facilitan la generación de páginas web dinámicas e interactivas.
  4. [Criterio d] Describe el proceso y los componentes involucrados en Spring Boot para manejar una solicitud HTTP del usuario y generar una respuesta dinámica en el lado del servidor.
  5. [Criterio e] Proporciona un ejemplo de cómo se puede utilizar Spring Boot para validar la entrada de un formulario en el servidor y cómo manejar y comunicar los errores de validación al cliente.
  6. [Criterio f] Explique cómo puede usar Thymeleaf junto con Spring Boot para cambiar dinámicamente el contenido de una página web basada en los datos del usuario.
  7. [Criterio g] Describa un escenario donde utilizaría Spring Data JPA en su aplicación Spring Boot y explique cómo este framework facilita la interacción con la base de datos.
  8. [Criterio g] Proporcione un código de ejemplo donde se muestre cómo se puede utilizar Spring Security para autenticar a los usuarios en una aplicación web desarrollada con Spring Boot.
