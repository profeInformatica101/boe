Rúbrica de Evaluación: RA2

Entrega CRUD de Producto con JSP y Servlets

Criterio de Evaluación Suspenso Aprobado Bien Notable Sobresaliente
a) Mecanismos de generación de páginas web No identifica ni utiliza mecanismos para generar páginas web Identifica mecanismos básicos sin implementarlos de forma adecuada Utiliza mecanismos de manera efectiva para generar algunas partes de las páginas web Utiliza mecanismos de manera eficiente en la mayoría de las partes de las páginas web Demostración excepcional en el uso de mecanismos para generar páginas web eficientemente
b) Tecnologías asociadas No identifica tecnologías asociadas Identifica, pero no aplica correctamente tecnologías asociadas Aplica algunas tecnologías asociadas correctamente Aplica correctamente la mayoría de las tecnologías asociadas Demuestra un uso excepcional de todas las tecnologías asociadas
c) Utilización de etiquetas para inclusión de código No utiliza etiquetas para inclusión de código Usa etiquetas para inclusión de código de forma inapropiada o errónea Usa etiquetas para inclusión de código correctamente en algunas partes Usa etiquetas para inclusión de código correctamente en la mayoría de las partes Usa etiquetas para inclusión de código de manera ejemplar en todas las partes
d) Reconocimiento de la sintaxis del lenguaje No reconoce la sintaxis del lenguaje utilizado Reconoce la sintaxis pero presenta varios errores en su uso Reconoce y aplica la sintaxis con errores mínimos Reconoce y aplica correctamente la sintaxis del lenguaje Demuestra un dominio excepcional de la sintaxis del lenguaje
e) Escritura y comprobación de sentencias simples No escribe ni comprueba sentencias Escribe y comprueba sentencias con varios errores Escribe y comprueba sentencias con algunos errores Escribe y comprueba la mayoría de las sentencias correctamente Escribe y comprueba sentencias de manera impecable
f) Utilización de directivas para modificar comportamiento No utiliza directivas Utiliza directivas de manera incorrecta o ineficiente Utiliza algunas directivas correctamente Utiliza la mayoría de las directivas de manera efectiva Utiliza directivas de manera experta y eficiente
g) Utilización de variables y operadores No utiliza variables ni operadores Utiliza variables y operadores con numerosos errores Utiliza variables y operadores con errores ocasionales Utiliza efectivamente variables y operadores en la mayoría de los casos Demuestra un uso experto y preciso de variables y operadores
h) Identificación de los ámbitos de utilización de las variables No identifica ámbitos de variables Identifica incorrectamente los ámbitos de variables en varias ocasiones Identifica y aplica los ámbitos de variables con algunos errores Identifica y aplica correctamente los ámbitos de variables en la mayoría de los casos Demostración excepcional en la identificación y aplicación de ámbitos de variables
