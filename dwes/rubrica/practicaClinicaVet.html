Práctica: Sistema de Gestión para una Clínica Veterinaria

Descripción

Desarrollar una aplicación web para gestionar las citas, los clientes y las mascotas de una clínica veterinaria.

Objetivos Específicos

  • Gestión de Usuarios: Crear roles diferenciados para clientes y administradores.
  • Gestión de Citas: Permitir la programación, cancelación y visualización de citas.
  • Gestión de Clientes y Mascotas: Permitir gestionar la información de las mascotas y los clientes.

Requisitos

Desarrollar la aplicación utilizando Spring Boot, implementando el mantenimiento del estado y autenticación de usuarios. Además:

  • Permitir la creación de cuentas de usuario y gestionar diferentes roles.
  • Los clientes pueden añadir, editar, o eliminar la información de sus mascotas.
  • Los administradores pueden gestionar las citas y acceder a la información de todos los clientes y sus mascotas.
  • Implementar el uso de sesiones para mantener el estado del usuario a lo largo de su interacción con la aplicación.
  • Utilizar cookies para personalizar la experiencia del usuario y/o recordar sus preferencias.
  • Integrar un sistema de autenticación de usuarios para proteger las cuentas de usuario y los datos sensibles.

Entregables

  • Código fuente de la aplicación.
  • Documentación técnica, que incluirá la arquitectura de la aplicación, el modelo de datos, y la justificación de las elecciones técnicas realizadas.
