Práctica: Desarrollo de Aplicaciones Web con Spring Boot y JPA

Resultado de Aprendizaje 6: Desarrolla aplicaciones web de acceso a almacenes de datos, aplicando medidas para mantener la seguridad y la integridad de la información.

Descripción de la Práctica

Desarrolle una aplicación web utilizando Spring Boot y JPA que permita a los usuarios gestionar información almacenada en una base de datos relacional. La aplicación debe permitir operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre la entidad "Producto" que debe tener al menos los campos "id", "nombre" y "precio". Además, asegure la integridad y seguridad de la información manipulada.

Criterios de Evaluación

  • a) Análisis de tecnologías para el acceso a datos.
  • b) Creación de aplicaciones que establezcan conexiones con bases de datos.
  • c) Recuperación de información almacenada en bases de datos.
  • d) Publicación de la información recuperada en aplicaciones web.
  • e) Uso de conjuntos de datos para almacenar la información.
  • f) Creación de aplicaciones web para actualización y eliminación de información.
  • g) Pruebas y documentación de las aplicaciones web.
